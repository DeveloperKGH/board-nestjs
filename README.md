## :closed_book: Description
NestJS 기반 간단한 게시판 기능 구현 프로젝트
* NestJS + Mongo + Typegoose 학습
* 게시물 등록, 게시물 조회 결과 Excel Download 기능 포함

## Running the app

```bash
# package.json
$ yarn install

# run
$ nest start

# watch mode
$ nest start --watch
```
